# 🚀 デジタル時間割アプリ

iPadやスマホからサクサク確認できる、自分専用のWeb時間割アプリです。
Streamlitを使用して、現在の校時を自動判定する機能を搭載しています。

## ✨ 特徴
- **自動校時判定**: アプリを開いた瞬間に、現在の時刻から「今何限目か」を自動で判別します。
- **サクサクUI**: キーボードが立ち上がらない選択式UIを採用し、ストレスなく操作できます。
- **週間一覧表示**: 下部には今週の授業を一覧で確認できるテーブルを配置。
- **日本時間対応**: サーバーの場所に関係なく、常に日本標準時で動作します。

## 🛠 使用技術
- **Language**: Python
- **Framework**: Streamlit
- **Library**: Pandas, Datetime
- **Infrastructure**: GitHub, Streamlit Cloud

## 📖 使い方
1. サイトにアクセスします。
2. 今日の曜日が自動で選択され、現在の授業が表示されます。
3. 他の時間割を確認したい場合は、曜日ボタンや校時ラジオボタンを切り替えてください。

## 🔧 カスタマイズ方法
`main.py` 内の `Subject` インスタンスや、各曜日の辞書（`monday_timetable` など）を書き換えることで、自分の時間割に変更可能です。
